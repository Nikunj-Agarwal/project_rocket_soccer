# Data directory

All runtime artifacts live under `data/` using the layout below. Path helpers are in [`src/data_layout.py`](../src/data_layout.py).  
**Logic & assumptions:** [`docs/README.md`](../docs/README.md).

## Layout

```
data/
├── dataset/
│   └── strike_dataset.npy          # Generated by: python -m src.data_generator
├── training/
│   └── training_log.csv            # Written by: python -m src.network
├── reports/
│   └── plots/                      # python scripts/generate_plots.py
│       ├── README.md               # index
│       ├── global/                 # training_curve, strikenet (any batch)
│       └── integration/{batch_id}/ # per test_main.py run
│           ├── README.md           # links seeds → tests/integration/{batch_id}/
│           ├── integration_summary.png
│           └── seed_{N}/
│               ├── trajectory.png
│               └── errors.png
├── tests/
│   ├── integration/                # python scripts/test_main.py
│   │   └── {YYYYMMDD_HHMMSS}/
│   │       ├── batch.log
│   │       └── seed_{N}/
│   │           ├── trajectory.csv
│   │           ├── simulation.mp4    # or simulation.gif
│   │           └── metadata.json
│   └── static/                     # python scripts/test_static_target.py
│       └── {YYYYMMDD_HHMMSS}/
│           ├── batch.log
│           ├── trajectory.csv
│           └── simulation.mp4
├── runs/
│   └── manual/                     # python src/main.py --save-video
│       └── manual_seed{N}_{timestamp}/
│           ├── trajectory.csv
│           ├── simulation.mp4
│           └── metadata.json
└── phase_archives/                 # Phase notes (documentation only)
```

## Commands (use `striker` conda env)

```powershell
conda activate striker

# Dataset + model
python -m src.data_generator --num_samples 100000
python -m src.network

# Tests (integration: 50 seeds by default, each → simulation.mp4)
python scripts/test_network.py
python scripts/test_static_target.py
python scripts/test_main.py
python scripts/test_main.py --seeds 10 21 32 43 54 7 14 28 35 42

# Manual run with video
python src/main.py --seed 10 --save-video

# Report plots (latest integration batch by default; use --batch for a specific run)
python scripts/generate_plots.py
python scripts/generate_plots.py --batch 20260521_022824
```

Direct Python (no `conda run`):

```powershell
C:\Users\nikun\miniconda3\envs\striker\python.exe scripts/generate_plots.py
```

## Video output

Simulations save **`simulation.mp4`** (in-memory frames, no `frames/` folder). Requires `imageio` and `imageio-ffmpeg` in the env:

```powershell
pip install imageio imageio-ffmpeg
```

If MP4 encoding fails, a **`simulation.gif`** is written instead.
